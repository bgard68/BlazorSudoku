### 3. **History of Sudoku** (Timeline)
Interactive timeline covering key milestones:

#### 1783 - Latin Squares
- **Leonhard Euler** created mathematical precursor
- n×n grids with symbols appearing once per row/column

#### 1979 - Modern Sudoku is Born
- **Howard Garns** (American architect) created "Number Place"
- Published in Dell Magazines
- Added 3×3 regional constraint

#### 1984 - Arrives in Japan
- Introduced by **Nikoli** puzzle publisher
- Named "S?ji wa dokushin ni kagiru" (????????)
- Shortened to **"Sudoku"** (??)
- Meaning: "number" (s?) + "single" (doku)

#### 2004 - Global Phenomenon
- **Wayne Gould** discovered puzzle in Japan
- Spent 6 years developing computer generator
- Convinced *The Times* (London) to publish
- Spread worldwide through newspapers

#### 2005-Present - Worldwide Popularity
- Global sensation: newspapers, books, apps
- World Sudoku Championships (started 2006)
- Millions of daily solvers
- Variants and competitive tournaments

### 4. **How to Play**
Four fundamental rules presented as cards:

1. **Row Rule**: Each row must contain 1-9 without repetition
2. **Column Rule**: Each column must contain 1-9 without repetition
3. **Box Rule**: Each 3×3 box must contain 1-9 without repetition
4. **Given Numbers**: Pre-filled cells are fixed, use logic for empty cells

### 5. **Solving Strategies** (Accordion)
Expandable sections for different skill levels:

#### Beginner: Naked Singles
- Find cells where only one number fits
- Check row, column, and box constraints

#### Beginner: Hidden Singles
- Find numbers that can only go in one position
- Within row, column, or box

#### Intermediate: Naked Pairs/Triples
- Two cells with same two candidates
- Eliminate those candidates from other cells
- Works for triples and quadruples too

#### Advanced: X-Wing
- Pattern across two rows/columns
- Candidates align in specific positions
- Eliminate from intersecting groups
